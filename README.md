# Orchidaceae v0.4.1 Release Candidate

An offline-first wellness companion built with plain HTML, CSS, JavaScript and PWA APIs. This release hardens the Phase 4 experience for its first real user without expanding the product scope.

## Use it

1. Extract the ZIP.
2. Open `index.html` in a modern browser.
3. Choose **Edit cycle** to add cycle information, if desired.

For installability and service-worker offline support, serve the extracted folder through HTTPS (or localhost), then use the browser’s **Add to Home Screen** option.

## Local privacy

Theme, display name, mood, water count, journal and cycle settings are stored only in the browser on that device. The app gracefully opens with defaults if local browser storage is unavailable or contains damaged data.

Cycle information is an estimate only and is not medical advice or contraception guidance.
